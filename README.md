# Behavioural Data Analysis: Stimulus-Aligned Chunks and Visualization

## Overview

This repository contains scripts for processing video tracking recordings aligned to stimulus onset and for visualizing the resulting data.

The workflow is divided into two stages:

1. **Data preparation** – extracting time windows ("chunks") of movement aligned to stimulus onset and storing them in a matrix.
2. **Visualization** – generating plots from the aligned data matrix and manual annotations.

## Repository Structure

```
.
├── data/                # raw or intermediate data files
├── scripts/             # analysis scripts
│   ├── create_chunks.py
```

## Step 1: Generate Stimulus-Aligned Data

Run the script:

```
create_chunks.py
```

This script:

* Loads the neural recording data
* Uses stimulus onset timestamps
* Extracts time windows around each stimulus
* Aligns them to stimulus onset
* Stores the result as a matrix

Typical output structure:

```
stim_aligned_matrix.npy
```

Matrix dimensions usually follow:

```
(trials × timepoints × channels)
```

or

```
(trials × timepoints)
```

depending on the dataset.

## Step 2: Generate Plots

After creating the aligned matrix, the following scripts generate visualizations.

### 1. Heatmap visualization

```
plot_heatmap.py
```

Displays the aligned responses as a 2D heatmap (e.g., trials vs time).

### 2. Average trace plots

```
plot_traces.py
```

Plots the average response across trials or channels aligned to stimulus onset.

### 3. Statistical or summary plots

```
plot_statistics.py
```

Computes summary metrics and generates additional visualizations such as:

* peak timing
* response magnitude
* trial distributions

## Typical Workflow

Run the scripts in the following order:

```
python create_chunks.py
python plot_heatmap.py
python plot_traces.py
python plot_statistics.py
```

## Requirements

Typical Python packages used:

```
pandas
```

```
pip install numpy matplotlib scipy pandas
```

## Notes

* The plotting scripts expect the aligned matrix generated by `create_chunks.py`.
* Ensure file paths inside the scripts point to the correct data directories.
* The scripts can be adapted to other stimulus-triggered neural recordings.
