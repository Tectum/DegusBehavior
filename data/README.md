# Stimulus-aligned response analysis

## overview

This folder contains original experimental data together with manually annotated responses and analysis scripts used to process and visualize the recordings.

1. **original data** containing the raw recordings
2. **manual annotations** stored in Excel files indicating the timing and classification of responses

## data description

### original data

The folder `data/raw_data` contains the original recordings used for the analysis.
These files represent the primary dataset from which stimulus-aligned response segments are extracted.

### manual annotations

The folder `data/annotations` contains Excel files with manual annotations of behavioral or neural responses.

These annotation files include information such as:

* stimulus onset times
* response timing
* manual classification of responses

The annotations are used by the analysis scripts to align the recordings to stimulus onset and to identify response events.
